# IDL и генерация кода

## Цели

- Жёстко зафиксировать границы между слоями.
- Упростить IPC‑вызовы через автогенерацию client/server‑стабов.
- Убрать ручное маршаллирование в критичных местах.

## Формат (план)

- Файл интерфейса: `*.defs`.
- Описание интерфейса в декларативном виде.
- Генератор создаёт:
  - client‑стабы,
  - server‑диспетчеры,
  - таблицу соответствия методов.

## Текущая реализация (v0)

- Добавлен минимальный генератор: `scripts/idl/idlgen.py`.
- Есть разбор `interface`/`rpc` и генерация header‑стабов.
- Поддерживаемые типы: `u32`, `u64`, `string`, `port`.
- Генерация пока не интегрирована с IPC/runtime.

## Инварианты

- Любой IPC‑интерфейс должен иметь `.defs`.
- Ручной маршаллинг — временная мера и должен быть вытеснен генерацией.
