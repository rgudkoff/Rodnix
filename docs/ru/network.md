# Сетевой стек

## Стратегия

- Сеть появится позже базовых подсистем.
- Первичный API — BSD‑сокеты с `AF_INET`.
- Порядок внедрения: минимальный loopback → UDP → TCP.

## Текущая реализация (v0)

- Есть минимальный loopback (очередь пакетов в памяти).
- Поддержка `net_loopback_send()`/`net_loopback_recv()` без протоколов.
- Добавлены базовые UDP‑сокеты (AF_INET + SOCK_DGRAM) только для loopback.
- Добавлены базовые STREAM‑сокеты (AF_INET + SOCK_STREAM) только для loopback, без полноценного TCP.
- TCP (реальный стек) и драйверы сетевых устройств отсутствуют.

## Инварианты

- До появления TCP должны быть стабильны UDP и loopback.
- Никакой "частичной" поддержки сокетов без четкого набора syscalls.

## Где смотреть в коде

- `kernel/net/net.c`, `kernel/net/net.h`.
