# Сетевой стек

## Стратегия

- Сеть появится позже базовых подсистем.
- Первичный API — BSD‑сокеты с `AF_INET`.
- Порядок внедрения: минимальный loopback → UDP → TCP.

## Инварианты

- До появления TCP должны быть стабильны UDP и loopback.
- Никакой "частичной" поддержки сокетов без четкого набора syscalls.

## Где смотреть в коде

- Планируется в `kernel/common` и `kernel/net` (каталог будет добавлен).
