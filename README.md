# RodNIX

Современное 64-битное ядро операционной системы, вдохновленное архитектурой XNU.

## Архитектура

RodNIX построен с учетом поддержки множества архитектур:
- **x86_64** (CISC) - Intel/AMD 64-битные процессоры
- **ARM64** (RISC) - ARM 64-битные процессоры
- **RISC-V64** (RISC) - RISC-V 64-битные процессоры

## Структура проекта

```
RodNIX/
├── osfmk/              # Основное ядро (Mach-подобная архитектура)
│   ├── mach/           # Архитектурно-независимые абстракции
│   ├── kern/           # Общие компоненты ядра
│   └── arch/           # Архитектурно-зависимые реализации
│       ├── x86_64/     # Реализация для x86_64
│       ├── arm64/      # Реализация для ARM64
│       └── riscv64/    # Реализация для RISC-V64
├── boot/               # Код загрузки
├── include/            # Заголовочные файлы
└── drivers/            # Драйверы устройств
```

## Особенности

- **64-битная архитектура**: Поддержка только 64-битных процессоров
- **Архитектурная абстракция**: Единый интерфейс для разных ISA
- **Модульность**: Четкое разделение архитектурно-зависимого и независимого кода
- **Расширяемость**: Легкое добавление поддержки новых архитектур
