# x86_64 - Реализация для архитектуры x86_64

Реализация ядра для процессоров Intel/AMD x86_64 (CISC архитектура).

## Особенности

- **CISC архитектура**: Complex Instruction Set Computer
- **64-битный режим**: Long Mode (x86-64)
- **4-уровневая страничная структура**: PML4 → PDPT → PD → PT
- **Канонические адреса**: Kernel в high-half (0xFFFFFFFF80000000)

## Компоненты

- `config.h` - конфигурация x86_64
- `types.h` - типы данных x86_64
- `interrupts.c` - обработка прерываний
- `memory.c` - управление памятью
- `cpu.c` - работа с процессором
- `boot.S` - код загрузки

## Регистры

x86_64 использует 16 64-битных регистров общего назначения:
- rax, rbx, rcx, rdx, rsi, rdi, rbp, rsp
- r8, r9, r10, r11, r12, r13, r14, r15

## Страничная структура

```
PML4 (512 entries) → PDPT (512 entries) → PD (512 entries) → PT (512 entries)
```

Размеры страниц:
- 4KB (обычные страницы)
- 2MB (large pages)
- 1GB (huge pages)
